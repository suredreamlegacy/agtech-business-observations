# 精准农业商业观察

## 简介

本项目是群体学习的一种尝试，主要关注遥感大数据与精准农业，近年来这方面发展很快，个人的力量是薄弱的，各类信息的质量又是参差不齐的。结果必然是：要么只能关注很小的方向，或者就是建立有效的信息过滤和评价方法。
